Bearbeitet:

WATTESTÄBCHEN IM OHR: Das Ohr reinigt sich von selbst. Ohrenschmalz fängt z.B. Dreck ab. Das Schmalz fließt dann langsam inklusive Dreck nach außen ab. Das Wattestäbchen schiebt den Schmalz Richtung Trommelfell. Das kann direkt beschädigt werden oder ein schmerzhafter Klumpen kann entstehen. Mit dem Wattestäbchen wird ein Juckreiz ausgelöst. Den möchte man wegkratzen - mit dem Wattestäbchen. Dadurch entsteht ein neuer Juckreiz. Gleichzeitig werden bestimmte Nerven stimuliert. Das fühlt sich so gut an, dass man es wiederholen möchte - Das kann süchtig machen.
